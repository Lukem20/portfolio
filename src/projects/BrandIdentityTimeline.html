Brand Identity Timeline

Role: Web Designer, Frontend Developer

Background

For a History of Design class, I had complete freedom to create a presentational project that only had to do with the history of some form of design. An additional, but optional challenge for the assignment was to make the presentation interactive somehow. At the time, I was trying to gain experience with scalable vector graphics on the web, so I challenged myself to create a "scrolly-telling" website. I wanted an element that was captivating to keep the audiences attention but didn't distract from the content presented. Although I was to make a website, I did't want it to stray too far from a traditional slide presentation.

Solution

My concept was to do a brief history of some of the oldest companies in the United States that have long standing and recognizable branding. Each section of the presentation web page would take up the full height of the screen so that I could move between them like slides. The height of the SVG was calcualted using the current scroll position, so that as I scrolled down to the next section, the brand of the next company was revealed. The turn-around was one week, so I kept the overall design minimal. In hindsight, this was a good idea too because I feel any more movement on the screen would have been distracting to the presentatinon content.
