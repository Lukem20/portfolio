The Interactive Textbook for
Geometric Linear Algebra

Role: Web Designer, Frontend Developer

Background

Linear algebra is essential to how computers represent three-dimensional objects and manipulate geometry in virtual space. This applies to many applications including video games, virtual reality, simulations, and more. The field of computer graphics is a complicated subject and most of the existing educational material on this topic is centered around this math.

Problem

Linear algebra is not a prerequisite to the graphics programming class at CSUMB. Furthermore, there is a lack of free online software that allows students to practice and interact with the applied math concepts. This shortage of material is detrimental to the learning process because the visual context of linear algebra is crucial to its application in computer graphics. It is also important that there is a source of free information available.

Link

graphics-textbook.vercel.app

Tools

Figma, Trello, Github

Skills

React, Figma, CSS, Three.js, React-Three Fiber

Project Type

School Project

Goal

My team and I wanted to have content that revolved around the application of the math concepts in a visual context, not necessarily about how the numbers behind the math worked. The reasoning behind this was to try and make each lesson approachable to students who had just been introduced to the topics for the first time. Each of the lessons would be interactive to solidify these applications with hopes to give students the tools to problem-solve when to use them.

Solution

My capstone team and I created a web application with React and React-three-fiber that acheives these main goals. The content explains concepts that are reinforced with animated and interactive sandbox activities. At the time of writing this, there are currently three chapters covering vectors, vector operations, and matrix transformations. The design uses typography to create an inviting atmosphere and a consistent look.
