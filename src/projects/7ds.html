The Seven Deadly Sins of
Social Media & Technology

Background

This is a frontend web project I made for an experimental typography class. The original task for was to take the seven deadly sins of a concept or phenomenon and create a tpyography book. I received permission to alter the assignment to create a website instead of a book so that I could practice making animations for the web. I am fascinated by the effects of social media and the technology companies that run them on society, so I decided that was my concept.

Role: Web Designer, Frontend Developer

Solution

Each page contains a header that uses an animation to try and encapsulate the character of the sin. I mostly used native CSS, but some of the sins also use GSAP. I decided to use Vite so that I could build my site after installing the animation library, but I ran into an issue with my page routing. Maybe this is naive, but I assumed I could route my pages using the paths to my HTML files, but Vite complained. I figured out how to properly write a Vite configuration file so that I could have HTML-like file based routing.I learned a lot about creating animation frames in CSS and also had good practice creating consistent layouts.
