Sisi's Barbershop

Background

Sisi wanted a website that provided each of the booking links for the barbers at her barbershop. Her previous experience with web consultants where high prices, and a miscommunication of project outcome and scope. They didn't need a newly built custom booking solution, just a website that they could publicly advertise which had all of the resident barbers existing booking links in one place. This made it so each barber could have autonomy in choosing their booking solution while also making it easy to update, add, or delete profiles from the website.

Role: Web Designer, Web Developer

Solution

Designed with the help of my brother, Cooper Moore, I created a satic website using core web technologies to develop a landing page for Sisi's Barbershop in Santa Rosa, California. The site is made using just HTML, CSS, and JavaScript, and in the future I would consider going back and refactoring the carousel to use no JavaScript either. The website boasts all 100's on Lighthouse and uses semantic HTML to be accessible by default.
