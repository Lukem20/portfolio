Three.js Sandbox

Role: Web Developer

Background

I first began learning graphics programming in 2022 when I found Discover Three.js, a free resource for learning the WebGL rendering library. I really enjoyed reading this book and practicing the concepts as I read along. Besides three.js, I learned more about asynchonous programming in JavaScript and how to better structure my code.

Solution

This website is the culmination of my experiments to practice using Three.js. In this project you will find a lot of geometries I made to practice using the basic mechanics of what the library is capable of including physically-based rendering, lighting, the animation loop, responsive sizing, texture mapping, and loading 3D models. The environment itself is honestly a bit of a mess with many geometries floating about.
