The Seven Deadly Sins of
Social Media & Technology

Role: Web Designer, Frontend Developer

Background

This is a frontend web project I made for an experimental typography class during university. The original task was to take the idea of seven deadly sins and apply that to a concept or phenomenon in order to create a tpyography book. I received permission to alter the assignment requirements to create a website instead of a book so that I could practice creating web animations. I am fascinated by the effects of social media and the technology companies that run them on society since they are a relatively new phenomena and we are just beginning to see their influence on important world events.

Solution

Each page contains a typographic header that uses an animation to try and encapsulate the character of the sin. I mostly used native CSS, but some of these animations also use the GSAP animation library. I used Vite so that I could build my site after installing the animation library, but I ran into an issue with page routing. Maybe this was naive, but I assumed I could route my pages using the paths to my HTML files, but Vite complained. I learned how to properly write a Vite configuration file so that I could have HTML-like file based routing. I learned a lot about creating animation frames in CSS and also had good practice creating consistent layouts.
